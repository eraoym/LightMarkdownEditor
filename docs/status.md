# 実装状況

> 最終更新: 2026-03-18（v0.3.0: PDF印刷・ページ番号・改ページ・プレビューテーマ追加）

---

## 実装済み

- Tauri v2 + React 19 + TypeScript のプロジェクト基盤
- Vite 7 によるフロントエンドビルド環境
- Tailwind CSS v4 の導入（`@tailwindcss/vite` プラグイン経由）
- OS のライト/ダークテーマへの自動連動（`prefers-color-scheme`）
- **ダークモード手動切替**（ヘッダーの ☀/🌙 ボタン、OS 設定を初期値として使用）
- Markdown テキストの入力 UI（`textarea`、モノスペースフォント）
- `pnpm tauri dev` でデスクトップウィンドウの起動確認
- TypeScript の型チェックエラーなし（`tsc --noEmit` 通過済み）
- Markdown プレビュー（`react-markdown` + `remark-gfm` + `@tailwindcss/typography`）
- EditMode / PreviewMode 切り替え（ヘッダーのトグルボタンで全画面表示を切替）
- ファイルの保存・読み込み（`tauri-plugin-fs` + `tauri-plugin-dialog`、Ctrl+S 対応）
- ウィンドウタイトル（開いているファイル名を反映、新規は `light-md`）
- **フォーマットツールバー**（Bold/Italic/H1-H3/箇条書き/番号リスト/コード、EditMode のみ表示）
- **キーボードショートカット**: `Ctrl+B`（太字）、`Ctrl+I`（斜体）、`Ctrl+Z`（アンドゥ）、`Ctrl+Y`（リドゥ）
- **Undo/Redo 履歴管理**（`useHistory`：最大200エントリ、タイピングは500msデバウンス、プログラム変更は即時コミット）
- **Tab インデント**: Tab でスペース2個挿入、Shift+Tab で削除
- **括弧・記号の自動補完**: `(`, `[`, `` ` `` に対応
- **ファイルタブ**: `useTabs` による多タブ対応（`[+]` 新規、`×` 閉じる、`●` 未保存、未保存時の確認ダイアログ）
- **Ctrl+Tab / Ctrl+Shift+Tab**: タブ切替ショートカット
- **エクスプローラーサイドバー**: ヘッダーの `☰` でトグル、フォルダ選択、ツリー遅延展開、ファイルクリックで新タブ/既存タブ切替
- **バイナリ非表示**: エクスプローラーからテキスト外ファイルを開くと「表示できません」メッセージをタブに表示
- **サイドバーサイズ可変**: ドラッグハンドルでリサイズ（160px〜480px）、localStorage 永続化
- **ウィンドウサイズ永続化**: アプリ再起動後もウィンドウサイズを復元
- **画像表示（相対パス）**: プレビューで相対パス画像を `readFile` + base64 で表示
- **mermaid 対応**: ` ```mermaid ` コードブロックをダイアグラムとして描画（ダーク/ライトテーマ連動）
- **シンタックスハイライト**: `highlight.js` によるコードブロックカラーリング（GitHub テーマ、ダーク/ライト連動）
- **設定画面**: ⚙ ボタンでモーダルを開き、エディタフォントサイズ・フォントファミリー・タブ幅を変更可能（localStorage 永続化）
- **プレビューテーマ切替**: GitHub / Minimal / Academic の 3 テーマを設定から選択可能
- **PDF 印刷**: プレビューモードの PDF ボタンから印刷ダイアログを呼び出し。ページ番号（CSS `@bottom-center`）自動表示、`---` による改ページ指定対応

---

- **アプリアイコン**: L（スカイブルー）と M（ホワイト）をジオメトリックに重ねたデザイン（`app-icon.svg` → `tauri icon` で全サイズ自動生成）
